Declaración de accesibilidad

Politica de accesibilidad y cumplimiento de los estándares

Movie Tracker tiene como objetivo hacer que todas las personas puedan acceder a la aplicación sin ningún tipo de barrera o impedimento. Para esto, tratamos de cumplir con todos los requisitos que se indican para el nivel AA de las Directrices de Accesibilidad dictadas por el W3C.

Idioma

Todas las páginas cuentan con la propiedad 

<html lang="es-AR">
lo cuál nos indica el idioma en el que está escrita la aplicación, para la correcta lectura por parte de los lectores de pantalla.

Estructura

Las páginas siguen el formato para cumplir con estos estándares, de modo que hemos respetado la estructura de encabezados consecuentes, para facilitar la navegación.

Tmbién, siguiendo con la línea del punto anterior, hemos separado HTML del CSS para que la información esté disponible siempre. El CSS empleado funciona de manera meramente decorativa.

Texto

El texto puede ser redimensionado por el usuario según su preferencia, hasta un 200% sin que por ello se pierda el contenido.

Enlaces

Todos los enlaces, incluso aquellos que estuvieran fuera de contexto, son comprensibles y claros.

Etiquetas

Todos los campos de texto están acompañados de su correspondiente label para una correcta asignación de la funcionalidad y que el usuario entienda cuál es el objetivo de ese campo.

Imágenes

Cuando es necesario mostrar una imagen, ésta está acompañada de un texto alternativo que proporciona información textual sobre la misma.

Dependencia de dispositivo

No se requiere un límite de tiempo específico para realizar las pulsaciones de las teclas. Todo el contenido es operable a través de un teclado.
